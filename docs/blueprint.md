# **App Name**: Nexus Student Marketplace

## Core Features:

- Listing Display: Display of the created student's listing, that includes a title, description, price, and optional images.
- Search and Filter Listings: Allow users to search for items and filter them by price and other categories.
- Student Profiles: Public facing profiles of students with space to show listed items, their college, a short bio and other relevant skills and interests.
- Profile Image Generation: Use a generative AI tool to generate a default profile picture if none is provided.
- Student Following: Allow students to follow each other to see listings that other students have posted.
- Listing Categorization: Automatic categorization of the user-posted content (Physical Products, Digital Products, Services, or Community/Collaboration).

## Style Guidelines:

- Primary color: Soft Lavender (#D0BFFF), evokes a sense of creativity, education, and trustworthiness relevant to the student environment.
- Background color: Light Gray (#F5F5F5), to ensure readability.
- Accent color: Muted Purple (#9479B9), offers a complementary contrast and highlights important UI elements such as buttons.
- Body and headline font: 'PT Sans', sans-serif; for both its modern look and a little warmth and personality.
- Simple, clean icons to represent product categories and other actions. Should be consistent with the color scheme.
- Clean, grid-based layout with clear sections for listings, profiles, and search.
